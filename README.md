# 직업상담 음성 분석 보조 툴

정부 고용지원 현장의 직업상담 실무자를 위한 브라우저 기반 업무 자동화 도구입니다.
긴 상담 음성을 입력받아 텍스트로 변환하고, 핵심 토픽/감정(불안도)/지원 니즈를 분석해 상담 일지 초안을 자동 생성합니다.

## 주요 기능

- 음성 입력 2가지
  - 파일 업로드 분석: `mp3`, `wav`, `m4a` 등
  - 실시간 녹음 분석: 브라우저 마이크 녹음 후 즉시 분석
- 자동 분석 결과
  - 주요 진로 토픽
  - 불안도/감정 상태
  - 지원 니즈
- 상담 일지 자동 생성
  - 실무형 요약문 자동 작성
  - 화자 혼합 녹취에서 `구직자 추출` 버튼으로 구직자 발화 우선 분리
  - 진로발달/진로상담 이론 반영 자동 해석
    - `Holland(RIASEC)` 관점
    - `SCCT`(사회인지 진로이론) 관점
    - `생애 진로발달` 관점(현 단계 탐색-준비-전환)
  - 구조화 출력 섹션 제공
    - `공식 상담일지(구조화)`
    - `진로발달/상담 이론 반영`
    - `심리/정서 분석`
    - `다음 회기 개입 전략`
  - 결과 텍스트 직접 수정 가능
  - 복사, TXT 다운로드, PDF 저장(브라우저 인쇄)

## 기술 구성

- Frontend: 순수 `HTML/CSS/JavaScript`
- 음성 → 텍스트: `transformers.js` + `Xenova/whisper-tiny` (브라우저 내 실행)
- 분석/요약: 브라우저 로컬 규칙 기반 분석 로직
- 백엔드: 없음

## 실행 방법

1. 저장소 클론

```bash
git clone https://github.com/pathcode04/straight.git
cd straight
```

2. `index.html`을 브라우저에서 열기

- Windows: 파일 더블클릭 또는 브라우저에 드래그
- 권장 브라우저: 최신 Chrome/Edge

3. 마이크 권한 허용 (실시간 녹음 사용 시)

## 사용 흐름

1. 상담일자/내담자 코드 입력
2. `파일 업로드 분석` 또는 `실시간 녹음 시작` 실행
3. 자동 변환된 텍스트 확인
4. 필요 시 `구직자 추출`로 구직자 발화만 분리
5. `텍스트 재분석`으로 결과 갱신 (구직자 추출본 우선 분석)
6. 분석 결과와 상담 일지 초안 수동 보정
7. `복사하기` 또는 `TXT 다운로드`/`PDF로 저장(인쇄)`

## 개인정보 및 보안 원칙

- 앱 자체는 외부 DB 저장 기능이 없습니다.
- 분석 결과는 브라우저 메모리에서 처리됩니다.
- 다만 최초 모델 로딩 시 모델 파일 다운로드를 위해 네트워크 연결이 필요할 수 있습니다.
- 민감 개인정보(주민등록번호, 상세 주소, 건강정보 등) 직접 입력은 지양하세요.

## 브라우저 호환/제약

- `MediaRecorder`, `getUserMedia`, `WebAudio API`를 지원하는 브라우저 필요
- 사내망/보안SW 정책으로 마이크 또는 CDN 접근이 차단되면 음성 분석이 실패할 수 있음
- 저사양 PC에서는 최초 모델 로딩과 긴 음성 분석 시간이 길어질 수 있음

## 문제 해결

- **마이크 권한 오류**: 브라우저 주소창 권한 설정에서 마이크 허용
- **음성 파일 분석 실패**: `wav` 또는 표준 `mp3`로 변환 후 재시도
- **모델 로딩 지연**: 네트워크 상태 확인 후 잠시 대기
- **결과 품질 보정**: 변환 텍스트를 직접 수정 후 `텍스트 재분석` 실행
- **화자 분리 정확도 보정**: `구직자 발화 추출본` 텍스트를 직접 수정 후 재분석

## 고지

- 본 도구의 결과는 자동 생성 초안입니다.
- 최종 상담 기록은 반드시 상담자의 전문 판단과 기관 지침에 따라 확정해야 합니다.
